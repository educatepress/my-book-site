## A+ Content（取締役承認後に作成）

以下の画像モジュールが効果的:
1. 著者紹介（白衣写真 + credentials）
2. 目次ハイライト（5章のビジュアル）
3. Before/After（読む前 vs 読んだ後）
4. 「こんな方に」チェックリスト
5. レビュー引用

**A+ Content は Amazon Seller Central → 広告 → A+ コンテンツ管理から作成。**

## アクション

1. **KDP 管理画面** (kdp.amazon.com) にログイン
2. 「本棚」→ 該当書籍 → 「...」→ 「書籍の詳細を編集」
3. タイトル、説明文、キーワードを上記に更新
4. JP 版と EN 版の両方を更新
5. 変更は24-48時間で反映
